# 🛡️ EmoShield — AI Cyberbullying Prevention

> **Built at Victor Hacks 2026** · Innovate. Create. Conquer.

EmoShield is a real-time AI-powered cyberbullying prevention tool that analyzes messages for emotional harm **before** they are sent — protecting children and teens from the lasting damage of online abuse.

---

## 🎯 The Problem

Cyberbullying affects **1 in 3 young people** worldwide. Most platforms only moderate content *after* harm is done. EmoShield intervenes **at the source** — the moment a harmful message is being typed.

---

## 💡 The Solution

EmoShield sits between the user and the send button. It uses **Claude AI** to:
- Detect toxic language, aggression, and emotional harm in real-time
- Understand **context and nuance** — not just keywords
- Generate a **kinder alternative** message automatically
- Deliver spoken warnings via **ElevenLabs voice alerts**
- Support **6 languages** with culturally aware analysis

---

## ✨ Features

| Feature | Description |
|---|---|
| 🧠 **Claude AI Analysis** | Real emotion detection — not keyword matching |
| 🎙️ **ElevenLabs Voice** | Spoken alerts in the user's language |
| 📊 **Safety Score** | 0–100 score showing how safe a message is |
| 🔁 **Auto-Rewrite** | Claude generates a kinder version of harmful messages |
| 🌍 **Multilingual** | English, Urdu, Arabic, Japanese, Spanish, French |
| 📈 **Dashboard** | Real scan history and weekly incident tracking |
| 🚫 **Auto-Block** | Severely toxic messages are blocked entirely |

---

## 🛠️ Tech Stack

- **Claude AI** (claude-sonnet-4) — Core emotion & toxicity analysis
- **ElevenLabs** — Voice alert engine
- **Web Speech API** — Browser-based text-to-speech fallback
- **Vanilla JS + HTML/CSS** — Zero dependencies, runs anywhere

---

## 🚀 How to Run

1. Download `emoshield_v3_HACKATHON.html`
2. Open it in any modern browser — **no installation needed**
3. Go to the **Settings** tab
4. Enter your Anthropic API key (`sk-ant-...`)
5. Type a message and hit **Scan with Claude AI**

> **No API key?** EmoShield still works in demo mode with pattern-based analysis.

---

## 🎬 Demo

1. Type a harmful message like *"you're so stupid nobody likes you"*
2. Hit **Scan with Claude AI**
3. Watch the Safety Score drop, emotions appear, and Claude suggest a kinder rewrite
4. The voice alert plays automatically via ElevenLabs
